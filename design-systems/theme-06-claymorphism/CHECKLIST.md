## 3. 접근성 점검

### 3.1 색 대비 (WCAG 2.1 — 본문 4.5:1, 그래픽/대형 3:1) — **계산 검증 완료**

라이트 테마 본문:
| 조합 | 비율 | 판정 |
|---|--:|:--:|
| 기본 텍스트 / surface | 15.89 | ✅ |
| muted 텍스트 / surface | 6.10 | ✅ |
| muted / surface-2 | 5.61 | ✅ |
| 텍스트 / 배경 | 14.02 | ✅ |

상태 솔리드 & 캔디 버튼(수정 후):
| 조합 | 비율 | 판정 |
|---|--:|:--:|
| 흰 / primary(violet-500) | 4.81 | ✅ |
| 흰 / danger(#d8341f) | 4.74 | ✅ |
| 흰 / info(sky-600) | 5.20 | ✅ |
| 흰 / success(mint-700) | 4.97 | ✅ |
| 짙은 / warning(yellow-400) | 9.91 | ✅ |
| btn-accent 짙은코랄(#4d1308) / coral-500 | 4.85 | ✅ |
| btn-mint 짙은민트(#053220) / mint-500 | 5.96 | ✅ |
| btn-yellow yellow-900 / yellow-400 | 4.95 | ✅ |

soft 뱃지 / 컬러 클레이 표면 텍스트(수정 후):
| 조합 | 비율 | 판정 |
|---|--:|:--:|
| violet-800/violet-100 · coral-800/coral-100 · mint-800/mint-100 · yellow-900/yellow-100 · sky-800/sky-100 | 6.3–8.7 | ✅ |
| clay-coral(#4d1308/coral-400) | 5.81 | ✅ |
| clay-mint · clay-yellow · clay-violet · clay-sky | 4.8–5.3 | ✅ |

다크 테마:
| 조합 | 비율 | 판정 |
|---|--:|:--:|
| 텍스트 / dark surface | 14.28 | ✅ |
| muted / dark surface | 8.69 | ✅ |
| dark-fg / primary·success·warning·info·danger (각 캔디 위) | 5.1–8.6 | ✅ |

> **수정 이력**: 초기 검증에서 `btn-accent`(흰/coral-500 = 3.07), `btn-mint`(2.38), `btn-sky`(3.62),
> `clay-coral` 텍스트(4.00)가 본문 기준 미달이었음 → 캔디 밝기를 유지하기 위해
> coral/mint 버튼은 **깊은 동색 텍스트**로, sky 버튼/표면은 **한 톤 깊은 파랑(sky-600)**으로 보정해
> 전부 4.5:1 이상으로 통과시킴.
> 아이콘 배경(`stat__icon--coral` 등 coral-900/coral-400 = 4.00)은 그래픽 요소로 3:1 기준 충족.

### 3.2 색 외 단서 (색맹 대응)
- ✅ 뱃지에 `.badge__dot` / 아이콘 병행, 상태 알림에 체크/경고/정보 아이콘, 인라인 노트에 아이콘.
- ✅ 정렬 가능 테이블 헤더는 화살표 아이콘으로 정렬 방향 표시.
